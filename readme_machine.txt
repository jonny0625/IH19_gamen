【システム名】
　新自動販売機システム

【概要】
　複数の飲料商品をカートに追加し購入することができます。
　現金、電子マネー、QRコードで決済可能です。
　広告、構内マップ機能があります。

【サイズ】
　350W×550H固定です。

【各画面概要】
　トップページはindex.htmlです。

　----------------------------------------------------------
　index.html 商品一覧画面(トップページ)
　----------
　商品一覧画面です。

　商品の画像または値段表示をクリック
　→モーダルで商品詳細画面が開きます。

　ーーーーーーーーーーーーーーーーーーー
　商品詳細画面(モーダル)

　値段やアレルギー情報などが表示されます。
　数量を指定し"カートに追加"をクリック
　→商品がカートに追加されます。
　ーーーーーーーーーーーーーーーーーーー

　"会計pay"をクリック
　→モーダルでカートに入った商品が表示され
　購入確認画面が開きます。

　ーーーーーーーーーーーーーーーーーーー
　購入確認画面(モーダル)

　カートに追加されている商品の一覧と
　"以上○○点でよろしいですか？"
　というメッセージが表示されます。

　"決定"をクリック
　→決済方法選択画面(kessai.html)に遷移します。
　ーーーーーーーーーーーーーーーーーーー

　"INFORMATION 駅構内マップ"をクリック
　→案内表示画面(info.html)へ遷移します。

　一定時間経過すると広告画面(ad.html)に遷移します。

　----------------------------------------------------------
　kessai.html 決済方法選択画面
　-----------
　支払方法を選択する画面です。

　"現金"をクリック
　→入金画面(nyuukin.html)に遷移します。

　"電子マネー"をクリック
　→モーダルで詳しい決済方法を選択する画面が開きます。
　詳しい決済方法をクリック
　→入金画面(nyuukin.html)に遷移します。

　"ＱＲコード決済"をクリック
　→モーダルで詳しい決済方法を選択する画面が開きます。
　詳しい決済方法をクリック
　→入金画面(nyuukin.html)に遷移します。

　----------------------------------------------------------
　nyuukin.html 入金画面
　------------
　現金/電子マネー/QRコードで入金する画面です。
　"支払う"をクリック
　→購入完了画面(complete.html)に遷移します。

　----------------------------------------------------------
　complete.html 購入完了画面
　-------------
　決済が完了したことを示す画面です。
　画面をクリックか時間経過
　→商品一覧画面(index.html)に遷移します。

　----------------------------------------------------------
　ad.html 広告画面
　-------
　広告を表示する画面です。
　画面をクリック
　→商品一覧画面(index.html)に遷移します。

　----------------------------------------------------------
　info.html 案内表示画面
　---------
　構内マップを表示する画面です。
　"自動販売機"をクリック
　→商品一覧画面(index.html)に遷移します。

　----------------------------------------------------------
